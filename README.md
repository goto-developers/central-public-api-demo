LogMeIn Central Public API demos
================================

This repository containts example scripts that demonstrate how to use [LogMeIn Central][central]'s [Public API][public-api-doc].

This whole repository is provided under the MIT license (read for details in the `LICENSE` file).

## User Synchronization

Directory: `sync-users`

Show how the Public API can be used to create, delete, change permissions of users and how to change group membership.

[central]: https://secure.logmein.com
[public-api-doc]: https://support.logmein.com/central/help/logmein-central-developer-center
